#### 使用说明

1、插电开机手机连接 GreenhouseConfig 热点；
2、如不自动跳转手动打开浏览器 http://192.168.4.1/wifiConfig  配置网络；
3、连接成功后 可访问 http://你路由器分配的ip/ 查看详情配置风扇、水泵的阈值等；
